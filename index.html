Publications

Distributed, Optimized control of a stochastic Non-consensus Opinion dynamic network

Bachelor's thesis

Distributed optimization of an opinion dynamic system composed of multiple agents, each with a linear behavior as their opinion. Implementing a decentralized, multi-agent reinforcement learning approach to address this problem optimally in addition to classical iterative optimization algorithms.

Autonomous Unmanned Aerial Vehicle Indoor Navigation Based on Deep Learning Approaches.

IEEE Xplore link

In this research two deep learning approaches for autonomous drone navigation in a Virtual Environment (the Unreal Engine) were introduced and compared. The first method uses a convolutional neural network to identify safe paths, while the second employs generative networks to create depth images.
